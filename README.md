# See More YT Videos

A Chrome extension that lets you customize the YouTube homepage video grid — show more videos per row, designed for laptop users.

## Features

- **Instant preview** — drag the slider and see changes applied to YouTube immediately, no page refresh needed
- **Persistent settings** — your preference syncs across devices via Chrome sync
- **SPA resilient** — works seamlessly as you navigate around YouTube, including back to the homepage
- **Lightweight** — vanilla JavaScript, zero dependencies

## How to Use

1. Click the extension icon in the Chrome toolbar
2. Drag the slider to choose how many videos per row (3–8)
3. That's it — YouTube updates in real time

## Install Manually

1. Download or clone this repository
2. Open `chrome://extensions` in Chrome
3. Enable **Developer mode** (top-right toggle)
4. Click **Load unpacked** and select the extension folder, or you can also drag the release zip file into the page directly.
